# Welcome to My Website! 👋

This is a simple Flask-powered website 🚀.

## Getting Started
### Requirements:
Python 3.x and Flask (dependencies in `pyproject.toml`).

### Installation:
1. Clone the repo
2. Install dependencies:
   ```bash
   uv sync
   ```

### Run Locally:
```bash
python app.py
```
Open [http://localhost:5000](http://localhost:5000) 🌐.

## Features
- Responsive design
- Clean interface

Happy browsing! 💻
